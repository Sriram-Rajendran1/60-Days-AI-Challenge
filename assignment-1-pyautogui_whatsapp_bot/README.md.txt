# WhatsApp Web Message Automation using PyAutoGUI

This script automates sending a message on **WhatsApp Web** using Python's `pyautogui` and `webbrowser` modules.  
It opens WhatsApp Web in your browser, searches for a specific contact or group, and sends a predefined message automatically.
